Lab 1: HTML Introduction and Practice

Lab Overview

In this lab, we practiced creating web pages using HTML5, focusing on basic structure, semantic elements, hyperlinks, images, and lists. The lab consisted of four exercises that progressively built our understanding of HTML fundamentals.

Exercise 1: First Web Page

We created a simple web page with hyperlinks, images, lists, header, footer, navigation, articles, sections, and figures. Key concepts learned:

  • Basic HTML document structure
  • Creating absolute and relative hyperlinks
  • Adding images with proper attributes
  • Using semantic HTML5 elements
  • Creating navigation menus with anchor links

Exercise 2: Share Your Travel Photos

We enhanced the first exercise by adding:

  • Logo images
  • Icon buttons for sharing and actions
  • Related photos section
  • Additional reviews
  • Footer navigation links

Exercise 3: Customer Relations Management Admin

This exercise focused on replacing generic <div> tags with semantic HTML5 elements such as <header>, <nav>, <main>, <section>, <aside>, <figure>, and <footer> to improve document structure and meaning.

Exercise 4: Art Store

We created an Art Store page from scratch, implementing all learned concepts including semantic markup, proper image handling, navigation, and structured content organization.

Key Takeaways

This lab reinforced the importance of:

  • Using semantic HTML5 elements for better structure
  • Validating HTML code for errors
  • Creating accessible web pages with proper alt text
  • Organizing content with appropriate semantic tags
  • Building consistent navigation across pages
