Lecture 3: Hyperlinks, Images, and Lists

Hyperlinks

Hyperlinks are created using the <a> (anchor) tag with the href attribute. Links can be:

  • Absolute links: Point to external websites (e.g., http://www.example.com)
  • Relative links: Point to files within the same website (e.g., page2.html)
  • Anchor links: Link to specific sections within the same page (e.g., #section1)

Images

Images are embedded using the <img> tag with required attributes:

  • src: Path to the image file
  • alt: Alternative text for accessibility
  • title: Tooltip text displayed on hover

Images can be wrapped in <figure> tags with <figcaption> for better semantic structure.

Lists

HTML supports three types of lists:

  • Unordered lists (<ul>): Bulleted lists for items without specific order
  • Ordered lists (<ol>): Numbered lists for items in sequence
  • Definition lists (<dl>): Lists of terms and their definitions

Best Practices

When creating links and images:

  • Always provide descriptive alt text for images
  • Use relative paths for internal links
  • Ensure all links are functional and accessible
  • Use semantic HTML5 elements like <nav> for navigation menus
