Lecture 2: HTML5 Structure and Semantic Elements

HTML5 Overview

HTML5 is the latest version of the HyperText Markup Language, introducing new semantic elements, form controls, and multimedia capabilities. It provides better structure and meaning to web documents compared to previous HTML versions.

Document Structure

Every HTML5 document should begin with the DOCTYPE declaration <!DOCTYPE html> and include the <html> element with a lang attribute. The document is divided into <head> (metadata) and <body> (content) sections.

Semantic HTML5 Elements

HTML5 introduces semantic elements that clearly define different parts of a web page:

  • <header>: Contains introductory content or navigation
  • <nav>: Defines navigation links
  • <main>: Contains the main content of the document
  • <article>: Represents independent, self-contained content
  • <section>: Groups related content with a heading
  • <aside>: Contains content tangentially related to main content
  • <footer>: Contains footer information

Benefits of Semantic Markup

Semantic HTML5 elements improve:

  • Accessibility for screen readers and assistive technologies
  • Search engine optimization (SEO)
  • Code maintainability and readability
  • Browser understanding of page structure
