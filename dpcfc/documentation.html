Getting Started Guide

Learn how to download, customize, and deploy your portfolio template

Step-by-Step Guide

1

Download ZIP File

Click the "Download Free" button on any template demo page. The ZIP file contains all necessary files including HTML, CSS, JavaScript, and placeholder images.

ZIP File Structure Preview
2

Open index.html in Browser

Extract the ZIP file to a folder on your computer. Double-click index.html to open it in your default web browser. This lets you preview the template locally.

your-template-folder/
├── index.html
├── css/
│ └── style.css
├── js/
│ └── main.js
└── images/
└── (your images)
3

Edit Images and Text in Code Editor

Open the template files in your favorite code editor (VS Code, Sublime Text, etc.). Replace placeholder images in the images folder and update text content in the HTML files.

  • Replace images: Keep the same file names or update paths in HTML
  • Update text: Find and replace placeholder text in HTML files
  • Customize colors: Modify CSS variables or color values in style.css
  • Add your content: Update sections like About, Portfolio, Contact
4

Upload to Hosting

Once you've customized your template, upload all files to a web hosting provider. You can use free hosting services or paid options depending on your needs.

Upload all files and folders to your hosting provider's public directory (usually public_html or www). Make sure to maintain the folder structure.

Frequently Asked Questions

Do I need coding experience to use these templates?

Basic HTML/CSS knowledge helps, but you can customize most content by simply editing text and replacing images. The templates are well-commented and easy to modify.

Can I use these templates for commercial projects?

Yes! All templates are free to use for personal and commercial projects. You can modify them as needed.

What code editor should I use?

We recommend VS Code (free), but any text editor works. Popular options include Sublime Text, Atom, or even Notepad++.

How do I change the colors and fonts?

Open the css/style.css file and look for color values (hex codes like #000000) and font-family declarations. Replace them with your preferred colors and fonts.

Do the templates work on mobile devices?

Yes! All templates are fully responsive and will automatically adapt to different screen sizes, including phones and tablets.
